# Member Features Guide

## Overview

This guide covers all features available to member users in the NSRC AMS.

## Dashboard

### Accessing the Dashboard
1. Login to your account
2. You will be automatically redirected to `/member/dashboard`
3. Or click "Dashboard" in the navigation menu

### Dashboard Components

**Quick Statistics**
- Total Sessions: Number of duty sessions completed
- Total Attendance Records: Count of attendance entries
- Total Duration: Total hours worked
- Active Session: Current ongoing session (if any)

**Recent Sessions**
- List of your 10 most recent duty sessions
- Session date, time, and duration
- Click to view session details

**Quick Actions**
- Time In: Start a new duty session
- View Attendance: See all attendance records
- View Performance: Check your metrics

## Attendance Management

### Time In/Out

**Starting a Session (Time In)**
1. Go to Member Dashboard
2. Click "Time In" button
3. System records the start time
4. Session is now active

**Ending a Session (Time Out)**
1. Go to Member Dashboard
2. Click "Time Out" button
3. System records the end time
4. Session duration is calculated automatically

**Viewing Attendance**
1. Go to Member > Attendance
2. View all your attendance records
3. Filter by date range if needed
4. See session details including duration

### Attendance Records

Each attendance record includes:
- Date and time
- Duration (hours and minutes)
- Session status (completed, ongoing)
- Notes (if any)

## Profile Management

### Accessing Profile
1. Click your profile icon (top right)
2. Select "Profile"
3. Or go to `/profile`

### Updating Profile Information

**Basic Information**
- Full Name
- Email Address
- Phone Number

**Personal Details**
- Birthdate
- Gender
- Current Address
- Home Address

**Educational Information**
- School/College
- Major/Course
- Year Level

**Professional Information**
- Primary Competency
- Skills
- Experience

**Emergency Contact**
- Emergency Contact Person
- Emergency Contact Number

### Changing Password
1. Go to Profile
2. Click "Change Password"
3. Enter current password
4. Enter new password (twice)
5. Click "Update Password"

**Password Requirements:**
- Minimum 8 characters
- At least one uppercase letter
- At least one lowercase letter
- At least one number
- At least one special character

## Two-Factor Authentication (2FA)

### Enabling 2FA
1. Go to Profile
2. Click "Enable Two-Factor Authentication"
3. Scan QR code with authenticator app (Google Authenticator, Authy, etc.)
4. Enter verification code
5. Save backup codes in a safe place
6. Click "Enable"

### Using 2FA
1. Login with email and password
2. Enter 6-digit code from authenticator app
3. Click "Verify"

### Backup Codes
- Generated when enabling 2FA
- Use if you lose access to authenticator app
- Each code can only be used once
- Store in a safe location

### Disabling 2FA
1. Go to Profile
2. Click "Disable Two-Factor Authentication"
3. Confirm action
4. 2FA is disabled

## Troubleshooting

### Cannot Time In/Out
- Check if you're logged in
- Verify your account status is active
- Try refreshing the page
- Contact admin if issue persists

### Performance Data Not Updating
- Wait a few minutes for data to sync
- Refresh the page
- Check if sessions are properly recorded
- Contact admin if data is missing

### Cannot Update Profile
- Verify all required fields are filled
- Check password requirements if changing password
- Try a different browser
- Contact admin if issue persists

For more help, see [Troubleshooting Guide](./TROUBLESHOOTING.md).
