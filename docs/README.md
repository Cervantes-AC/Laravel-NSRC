## System Overview

The NSRC AMS is a Laravel-based web application that provides:

- **User Authentication**: Secure login with 2FA support
- **Attendance Tracking**: Real-time time in/out functionality
- **Performance Analytics**: Detailed metrics and rankings
- **Admin Dashboard**: Comprehensive management tools
- **Reporting**: Generate and export various reports
- **Audit Logging**: Track all system activities
- **Data Import/Export**: Bulk operations for data management
